## Decisions & Rationale

| Decision | Options Considered | Chosen | Rationale |
|----------|-------------------|--------|-----------|
| Date comparison | String comparison / Timestamp | Timestamp | More reliable across date formats |
| Error handling | set -e / set -euo pipefail | set -euo pipefail | Stricter, catches more issues |
| Protected dirs | Config-based / Hardcoded | Hardcoded | Simpler, retrospective/knowledge-base always permanent |

## Lessons Learned

- Always verify archive success before deleting files
