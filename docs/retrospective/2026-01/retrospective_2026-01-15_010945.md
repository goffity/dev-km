### Key Implementation

**Auto-assign with proper feedback:**
```bash
if gh issue edit "$ISSUE_NUMBER" --add-assignee @me 2>/dev/null; then
    echo "✓ Assigned issue #$ISSUE_NUMBER to you"
else
    echo "→ Auto-assign skipped (already assigned or no permission)"
fi
```

**Auto-start polling:**
```bash
if [[ -f "$PID_FILE" ]] && ps -p "$(cat "$PID_FILE")" > /dev/null 2>&1; then
    echo "✓ PR polling daemon already running"
else
    "${SKILL_DIR}/scripts/pr-review-poll-start.sh" --interval 300 2>/dev/null || true
fi
```
