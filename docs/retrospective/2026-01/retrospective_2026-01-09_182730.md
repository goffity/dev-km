## Context: Before

- **Problem**: ตัวแปรใน HEREDOC ไม่ expand - ใช้ `<<'EOF'` ซึ่งป้องกัน variable expansion
- **Existing Behavior**: `/focus` skill สร้างไฟล์ `docs/current.md` แต่ตัวแปรถูกเขียนเป็น literal text (เช่น `$TASK_INPUT` แทนที่จะเป็นค่าจริง)
- **Why Change**: ผู้ใช้ไม่สามารถดู current focus ที่ถูกต้องได้
- **Metrics**: Variables ไม่ expand 100%

---

## Context: After

- **Solution**: เปลี่ยนจาก `<<'EOF'` เป็น `<<EOF` และเพิ่ม feature branch enforcement
- **New Behavior**:
  1. ตัวแปรใน HEREDOC expand ได้ถูกต้อง
  2. บังคับให้สร้าง feature branch จาก main/default branch
- **Improvements**:
  - Current focus แสดงค่าที่ถูกต้อง
  - Workflow ปลอดภัยขึ้นด้วย feature branch
- **Metrics**: Variables expand 100%, feature branch enforced
