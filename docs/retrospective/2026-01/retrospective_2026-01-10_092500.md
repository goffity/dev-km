## Lessons Learned

- **JSON Safety**: Always use `jq --arg` for dynamic JSON construction, never string interpolation
- **Jira API**: Uses workflow transitions, not direct status changes
- **PR Review**: Copilot catches documentation inconsistencies well
- **Hybrid Integration**: Need clear paths for users who use multiple issue trackers
