### Key Implementation Details

1. **Sanitization for AppleScript**:
```bash
sanitize_for_applescript() {
    local str="$1"
    str="${str//\\/\\\\}"
    str="${str//\"/\\\"}"
    echo "$str"
}
```

2. **Process substitution to avoid subshell**:
```bash
while read -r pr; do
    # process PR
done < <(echo "$prs" | jq -c '.')
```

3. **Array for arguments with spaces**:
```bash
ARGS=()
ARGS+=("$1")
nohup "$POLL_SCRIPT" "${ARGS[@]}" >> "$LOG_FILE" 2>&1 &
```

## Lessons Learned

- Sanitize all user-controllable strings before passing to osascript
- Use arrays instead of strings for shell arguments to handle spaces properly
- Unused variables should be removed to keep code clean
