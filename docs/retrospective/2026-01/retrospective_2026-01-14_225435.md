## Lessons Learned

- **GitHub Milestones API**: ใช้ `gh api repos/:owner/:repo/milestones` สร้าง milestones ได้ง่าย
- **Discussion Comments**: ต้องใช้ GraphQL mutation `addDiscussionComment` ไม่ใช่ REST API
- **Multi-terminal Support**: Warp, Hyper, iTerm2 มี APIs ต่างกัน ควร implement แบบ layered
