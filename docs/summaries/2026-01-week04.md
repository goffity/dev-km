## Lessons Learned

- Claude Code discovers commands from `.claude/commands/`, not `assets/commands/`
- `gh api` returns newline-delimited JSON, `gh pr list` returns array - normalize input
- jq can handle complex text processing but regex support is limited
- GitHub doesn't always auto-close issues with multiple "Fixes #N" references
